Criptografia Simétrica e Assimétrica

Criptografia é uma maneira de proteger informações sensíveis onde que somente a pessoa dona desse conteúdo consiga vê-la. Essas informações são ocultadas a partir de uma série de técnicas na qual essas informações se tornam ilegiveis para qualquer um que não tenha uma chave de acesso. Essa chave de acesso pode ser de dois tipos, a Chave Simétrica e a Chave Assimétrica.

Para o texto poder ser criptografado, ele não pode conter:

  • Números, como 1, 2, 3, ...
  • Caracteres especiais, como ç, ~, /, ã, ...
